Exemplo de calculo do IMC, integrando HTML com PHP

Fórmula do IMC:
IMC = peso / (altura x altura)
ou
IMC = peso / (altura ^ 2)

Tabela de Valores:
Valor       Descricao
< 18,5      Peso Baixo
18,5 — 24,9 Peso Normal
25,0 — 29,9 Sobrepeso
30,0 — 34,9 Obesidade (Grau I)
35,0 — 39,9 Obesidade Severa (Grau II)
> 40,0      Obesidade Marbida (Grau III)
